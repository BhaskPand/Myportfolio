# bhaskarpandit.in

Personal portfolio of **Bhaskar Pandit M N** — AI Engineer & Full-Stack Developer.

Live at **[bhaskarpandit.in](https://bhaskarpandit.in)** (GitHub Pages).

## Stack

Zero-build static site — no framework, no bundler, no CDN runtime.

- `index.html` — semantic single-page layout, SEO meta + Open Graph + JSON-LD
- `styles.css` — hand-written design system (CSS custom properties, dark/light themes)
- `script.js` — dependency-free interactions (theme toggle, typewriter, scroll reveal, scrollspy)

## Develop

Open `index.html` in a browser, or serve locally:

```sh
python -m http.server 8000
```

Deploys automatically via GitHub Pages on push to `main`.
