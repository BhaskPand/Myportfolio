03 — Projects

Featured · Live LLM-Powered

ExamsAI — AI Study Planner for Competitive Exams

Problem

Exam aspirants juggle multiple competitive exams with generic, one-size-fits-all prep schedules — and no way to adapt plans to their own pace and syllabus.

Solution

A full-stack platform that generates personalized study plans using GPT-3.5 — prompt-engineered LLM pipelines turn a student's exams, dates and inputs into an actionable schedule.

Architecture

React + Material UI frontend on Vercel, Flask REST API on Render, SQLite persistence, OpenAI API integration — a cleanly separated, independently deployed client/server system.

Impact

Deployed and live — turns hours of manual schedule planning into a single prompt-driven flow, with plans that adapt to each student's exams, dates and pace.

LokSuraksha — Citizen Safety Platform

Problem

Civic hazards — broken infrastructure, unsafe zones — go unreported because there's no low-friction channel between citizens and the authorities who can act.

Solution

An end-to-end reporting system: a mobile app for GPS-tagged hazard reports with photos, and a web admin dashboard for authorities to triage, track and resolve them.

Architecture

React Native (Expo) mobile client, Node.js backend, and an admin web dashboard — one API serving two clients, built and led at KLS Tech Solutions.

Impact

A working civic-tech pipeline from citizen to authority: report in seconds, triage in one dashboard — built and led as a startup product, not a class project.
